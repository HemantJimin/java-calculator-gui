# Java Calculator GUI

## Description
A simple yet functional calculator application with a graphical user interface built using Java Swing. This project demonstrates the implementation of basic GUI components and event handling in Java.

## Features
- ✅ Basic arithmetic operations:
  - Addition (+)
  - Subtraction (-)
  - Multiplication (*)
  - Division (/)
- ✅ Decimal number support
- ✅ Clear (C) button to reset calculator
- ✅ Delete (Del) button to remove last digit
- ✅ User-friendly GUI with button layout
- ✅ Error handling for division by zero

## Technologies Used
- **Language:** Java
- **GUI Framework:** Java Swing
- **Layout Manager:** GridLayout

## How to Run

### Prerequisites
- Java Development Kit (JDK) 8 or higher installed on your system
- Basic knowledge of command line/terminal

### Steps to Run:

1. Clone the repository:
   ```bash
   git clone https://github.com/HemantJimin/java-calculator-gui.git
   cd java-calculator-gui
   ```

2. Compile the Java file:
   ```bash
   javac src/Calculator.java
   ```

3. Run the application:
   ```bash
   java -cp src Calculator
   ```

## Usage
1. Launch the calculator application
2. Click on number buttons (0-9) to enter numbers
3. Click on operator buttons (+, -, *, /) to select operation
4. Click on the equals button (=) to see the result
5. Use the decimal button (.) for decimal numbers
6. Use Clear (C) to reset the calculator
7. Use Delete (Del) to remove the last digit
